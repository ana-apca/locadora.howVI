# 📱 Locadora de Veículos — App Android

Aplicativo Android para gerenciamento de locadora de veículos, desenvolvido como **Projeto Integrador do 3º Semestre de ADS**.

## Sobre o Projeto

App mobile nativo para Android que permite o gerenciamento completo de uma locadora de veículos, incluindo cadastro, consulta e reserva de veículos diretamente pelo celular.

## Tecnologias

- **Plataforma:** Android
- **Linguagem:** Java
- **IDE:** Android Studio

## Funcionalidades

- Cadastro de veículos e clientes
- Sistema de reservas
- Consulta de disponibilidade
- Interface mobile nativa

## Como Executar

1. Clone o repositório
2. Extraia o conteúdo de `locadoravei.zip`
3. Abra o projeto no Android Studio
4. Execute em um emulador ou dispositivo físico
